## 使用yoman 创建脚手架 

- npm install -g yo
- npm install -g generator-generator (Yoman generator generating a Yoman generator)
- yo generator 创建脚手架的基础框架
- 创建模板，编写脚本
- npm link 链接到本地调试
- yo name 
- npm publish 发布
